# CHARISHMA S Portfolio Website

This is a fully responsive personal portfolio website for CHARISHMA S, a fresher software developer and aspiring AI engineer.

The portfolio is built using only HTML, CSS, and JavaScript. It is suitable for recruiters, internship applications, entry-level software roles, AI/ML roles, robotics roles, and engineering project showcases.

## Features

- Modern responsive design
- Sticky navigation bar
- Mobile hamburger menu
- Smooth scrolling
- Scroll reveal animations
- Active navigation highlight
- Hero section with profile image
- Resume download button
- About section
- Categorized skills with progress bars
- Project cards with images and technology tags
- Education timeline
- Contact form using mailto
- Footer with copyright
- SEO metadata

## How to Run Locally

Open the project folder in VS Code.

Then open:

```text
index.html
```

Right-click and choose:

```text
Open with Live Server
```

Or open `index.html` directly in your browser.

## How to Replace Profile Photo

Replace this file:

```text
assets/images/profile.jpg
```

Use the same file name:

```text
profile.jpg
```

## How to Update Resume

Replace this file:

```text
assets/images/Resume.pdf
```

Use the same file name:

```text
Resume.pdf
```

## Deployment

### GitHub Pages

1. Create a GitHub repository named `portfolio-website`.
2. Upload all files.
3. Go to repository Settings.
4. Click Pages.
5. Select the `main` branch.
6. Save.

Free domain example:

```text
https://username.github.io/portfolio-website
```

### Netlify

1. Go to Netlify.
2. Click Add New Site.
3. Drag and drop this project folder.
4. Deploy.

Free domain example:

```text
https://your-site-name.netlify.app
```

### Vercel

1. Go to Vercel.
2. Import the GitHub repository.
3. Click Deploy.

Free domain example:

```text
https://your-site-name.vercel.app
```
